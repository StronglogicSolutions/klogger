# KLogger

Async fmt logger inspired by spdlog and g3log for C++20 and up, but simpler and with coloured logging. Object-based (no macros).

## Sinks
- File sink
- Console sink

## Log Levels
- trace
- debug
- info
- warn
- error
- fatal
- silent

## Dependencies
- FMT: https://github.com/fmtlib/fmt


That's it, that's all
